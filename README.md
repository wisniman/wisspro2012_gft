wisspro2012_gft
===============

Wissenschaftliches Projekt (BHTB MIM WS12 [01]): Google Fusion Tables & API

Wisniewski + Boehm: Google Fusion Tables
08.10. Exposee (ein Absatz); Idee gut

Exposee:
Der nahenden Winterzeit eingedenk wuerden wir gern ein "Weihnachtsmarkt-Datencockpit" entwerfen, wobei verschiedene Informationen rund um die Berliner Weihnachtsmärkte gesammelt und grafisch aufbereitet werden. Denkbar wären neben Informationen wie Ort, Öffnungszeiten, Verteilung auf Bezirke auch detailliertere Infos wie z.B. durchschnittliche Glühweinpreise oder Anzahl der Fahrgeschäfte, etc. Dazu würden wir in einem ersten Schritt gern ein Webportal entwerfen, dass bestehende Daten (Adressen usw.) mithilfe der Fusion Tables aufbereitet und verschiedene Darstellungsmöglichkeiten zur Auswahl bietet. In einem weiterführenden Schritt würden wir dann ein (Web-)Interface zum Einpflegen neuer Daten (z.B. mithilfe des Open Data Kits) konzipieren. Je nach Fortschritt in der Umsetzung sind weitere Add-Ons denkbar, die uns nach Einarbeitung in die Materie als sinnvoll erscheinen.
